**Test 2 — Schema completeness check:**
Verify every required field is present on every result:
```bash
cat /tmp/digest-t1-stdout.txt | node -e "
const d = JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));

let missing = 0;
(d.results||[]).forEach((r,i) => {
```

**Test 3 — Today-only filter:**
Verify that the date field on results is a time-only string (h:mm AM/PM or Day h:mm AM/PM format), confirming only today's messages are included (per D-06: today's messages show time-only, not a full date):
```bash
cat /tmp/digest-t1-stdout.txt | node -e "
const d = JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
const TIME_RE = /^(Mon|Tue|Wed|Thu|Fri|Sat|Sun)?\s*\d{1,2}:\d{2}\s*[AP]M$/i;
(d.results||[]).forEach((r,i) => {
  if (r.date && !TIME_RE.test((r.date||'').trim())) {
    console.warn('result', i, 'has non-time date:', r.date, '-- may not be today');
  }
});
console.log('date format check complete');
"
```

Note: Some results may have day-prefixed times like "Thu 4:00 PM" for messages from earlier today (same-week format). These are still valid today messages. The check above warns but does not fail on day-prefixed times.

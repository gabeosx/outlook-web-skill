**Step 1: Navigate to calendar and capture full ARIA tree.**

Build and execute this exploration batch using `runBatch` from `lib/run.js` (or invoke agent-browser directly via CLI):

```javascript
const CALENDAR_URL = process.env.OUTLOOK_BASE_URL.replace(/\/mail\/?.*$/, '/calendar/');

const explorationBatch = [
  ['open', CALENDAR_URL],
  ['wait', '5000'],
  ['get', 'url'],
  ['eval', "JSON.stringify(Array.from(document.querySelectorAll('[role=option],[role=gridcell],[role=row],[role=listitem],[role=button]')).slice(0, 10).map(el => ({ tagName: el.tagName, role: el.getAttribute('role'), id: el.id, dataset: Object.keys(el.dataset), ariaLabel: el.getAttribute('aria-label'), className: el.className.toString().slice(0, 80) })))"],
  ['snapshot'],
];
```

Run this via: `node -e "require('./lib/run').runBatch(...)"`  or create a temporary exploration script.
